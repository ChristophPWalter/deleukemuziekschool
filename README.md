Bij De Leuke Muziekschool leer je een instrument spelen aan de hand van de muziek die jij leuk vindt.
Jouw muzieksmaak en ambitie staan centraal in de lessen.
"Met een doelgerichte aanpak om jou zo snel mogelijk je lievelingsliedjes te leren spelen brengen we je de techniek en theorie bij die je nodig hebt, zonder dat het saai wordt."
De focus ligt op:
Piano voor beginners
Gitaar voor beginners
Ukulele voor beginners
